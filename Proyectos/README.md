# Proyectos

Una carpeta por cada proyecto o esfuerzo activo. Cada uno tiene su propio `INDEX.md`
(creado desde la plantilla en `../Templates/`).

Cuando le digas a tu asistente que vas a empezar algo nuevo, él crea la carpeta aquí
automáticamente. Cuando un proyecto termina o se cancela, se mueve a `_archivo/`.
